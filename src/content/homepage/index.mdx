# Welcome to Mantik

### Choose Your Learning Path

Welcome to Mantik [(pronounced "mun-tick" or "mun-tiq")](https://youtu.be/BioAlyEcxxk)! This comprehensive training platform is designed to help you master programming concepts for robotics competitions. Select the training path that best fits your needs and experience level.

### About Each Section:

- **Java Training:** Master Java programming fundamentals: variables, data types, control structures, loops, object-oriented programming (classes, objects, methods), exception handling, and version control with Git and GitHub for team collaboration.
- **FTC Training:** Comprehensive FIRST Tech Challenge programming: OnBot Java development (hardware control, sensors, teleop, autonomous), Android Studio advanced programming (PID control, odometry, path planning, motion profiling), and professional debugging practices.
- **FRC Training:** Master comprehensive FRC programming: environment setup, command-based architecture, CTRE and REV hardware integration, control theory (PID, motion profiling, feedforward), pose estimation, path planning, and advanced tools like PathPlanner and AdvantageKit for competitive robotics.
- **Competitive Training:** Develop algorithmic problem-solving skills: data structures (arrays, trees, graphs, hash tables), algorithms (sorting, searching, dynamic programming, greedy), Big O analysis, interview preparation, and competition strategies for coding contests.

### How to Navigate:

Click on any of the training sections above to explore that area. Each section contains multiple lessons that you can access through the sidebar navigation. We recommend starting with Java Training if you're new to programming, or jumping directly to your specific competition type if you have prior experience.
